## Installation

### Set up git

```
cd $HOME
git init
git remote add origin https://github.com/willclarktech/dotfiles.git
git fetch
git checkout origin/master
```

### Copy/create files not held in VCS

See list at the end of the `.gitignore` file.

### Run install script

```
$HOME/.dotfiles/install.sh
```
